Query-Crafting DoS Threats Against Internet DNS

Recent DoS attacks against DNS (e.g., Dyn) show that disrupting DNS availability cascades into widespread service outages. This work studies a distinct DoS vectorâ€”query-crafting attacks, where adversaries generate specially crafted domain-name payloads to maximize per-query cost rather than relying solely on flooding or amplification. By modeling DNS with a state diagram, the authors identify threat vectors involving invalid, random, and mixed-structure domain names that force extra resolver processing. Internet measurements confirm that such queries increase resolver communication and latency. The authors propose defense features tailored to detecting query-crafting behaviors and evaluate a prototype on CloudLab.

Distributed Security Network Functions against Botnet Attacks

Botnets continue to dominate network attacks due to their distributed nature, scale, and diverse techniques, limiting effectiveness of centralized defenses. This paper proposes a distributed security architecture leveraging SDN/NFV and machine learning to counter large-scale disruptive botnet attacks. Distributed network functions monitor protocol-specific traffic, detect known attacks, and forward essential real-time data to an SDN controller. The controller performs advanced machine-learning analysis to detect large-scale distributed attacks and dynamically updates flow rules or enforces routing controls. Evaluations show system efficiently detects both local and widespread botnet attacks, providing an effective, scalable defense mechanism.
